Política de Privacidade

Esta Política de Privacidade descreve como as informações são coletadas, utilizadas e protegidas no Sistema de Triagem Jurídica, um projeto demonstrativo desenvolvido para fins acadêmicos e técnicos.

1. Finalidade do Sistema

O sistema tem finalidade exclusivamente demonstrativa, sendo utilizado para testes de integração com a API oficial do WhatsApp. Não se trata de plataforma comercial ativa.

2. Dados Coletados

Durante a interação podem ser coletados:

  • Nome completo
  • Número de telefone
  • Cidade/Estado
  • Área jurídica informada
  • Descrição resumida do caso

3. Uso das Informações

Os dados são utilizados exclusivamente para fins de triagem e eventual contato posterior pelo responsável pelo projeto. Não há compartilhamento com terceiros nem comercialização de dados.

4. Armazenamento e Segurança

As informações podem ser armazenadas temporariamente em ambiente controlado, com acesso restrito ao desenvolvedor responsável.

5. Direitos do Titular

O usuário pode solicitar a exclusão de seus dados a qualquer momento por meio do mesmo canal utilizado para a comunicação inicial.

6. Conformidade Legal

Este projeto observa os princípios da Lei Geral de Proteção de Dados (Lei nº 13.709/2018 – LGPD), respeitando a privacidade e proteção das informações pessoais.

Termos de Serviço

1. Natureza do Serviço

O Sistema de Triagem Jurídica é um projeto demonstrativo, desenvolvido para fins acadêmicos e testes técnicos. Não substitui consulta jurídica formal.

2. Responsabilidade

As informações fornecidas pelo usuário são de sua responsabilidade. O sistema não realiza análise jurídica automática nem emite parecer legal.

3. Limitação de Garantia

Por se tratar de ambiente de testes, não há garantia de disponibilidade contínua, estabilidade do serviço ou armazenamento permanente das informações.

4. Aceitação

Ao interagir com o sistema, o usuário declara estar ciente de que se trata de ambiente demonstrativo e concorda com a coleta e uso das informações conforme descrito na Política de Privacidade.
